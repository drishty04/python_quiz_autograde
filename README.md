# Python Quiz Auto-grade System

## How to Submit

1. Fork this repository.
2. Open `quizzes/quiz1.ipynb` in Jupyter Notebook.
3. Fill your details and answers.
4. Copy your completed notebook to `submissions/YOURID_quiz1.ipynb`.
5. Commit and push.
6. Open a Pull Request.

GitHub Actions will automatically grade your notebook and comment the score.
